# v1.1.0

2026 年 1 月 28 日

## 概览
这是一次重大版本发布，引入了多项强大的新功能 同时还包含显著的性能提升、系统优化与更强的可观测性能力。

---

## SDK

### 新功能

#### 运行时环境（Runtime Environment）
**[运行时环境参考文档](../References/Python%20SDK%20References/runtime-env.md)**

- 支持 Python/Node.js 运行时环境，易于使用
- 支持自定义运行时环境，易于实现
- 与沙箱代理（Sandbox Agent）集成

#### Model Service（实验性）
**[Model Service 参考文档](../References/Python%20SDK%20References/model-service.md)**

- 处理 Agent 与 LLM 推理服务之间的模型调用通信
- Local 模式：Agent 与 Roll 运行时之间的文件通信, 基于文件的通信协议，实现请求-响应机制
- 与 RockAgent 集成，自动管理生命周期
- SDK支持 anti_call_llm，用于代理Agent请求


#### ROCK Agent（实验性）
**[ROCK Agent 参考文档](../References/Python%20SDK%20References/rock-agent.md)**

ROCK 框架核心 Agent 实现：
- 提供完整的 Agent 生命周期管理
- 简洁 API：`sandbox.agent.install()` 和 `sandbox.agent.run(prompt)`
- 支持自定义 Agent 安装与运行
- 自动管理 ModelService 生命周期
- 自动初始化 RuntimeEnv


#### 沙箱加速 (Sandbox Speedup)
**[沙箱加速参考文档](../References/Python%20SDK%20References/python_sdk.md)**

提供针对沙箱环境的网络优化与镜像配置能力：
- 支持一键配置 APT 公共镜像源
- 支持 PIP 镜像源加速（自动处理 HTTP/HTTPS 协议及信任主机配置）
- 通过 Hosts 映射实现 GitHub 访问加速
- 采用非侵入式配置，仅优化源地址而不强制执行安装
- 基于策略（Strategy）的 URL 自动解析与标准化逻辑

#### FileSystem
**[FileSystem 参考文档](../References/Python%20SDK%20References/file_system.md)**

沙箱环境文件系统操作接口：
- `chown`：修改文件/目录所有者，支持递归操作
- `chmod`：修改文件/目录权限，支持递归操作
- `upload_dir`：通过 tar.gz 上传本地目录到沙箱

#### Deploy
**[Deploy 参考文档](../References/Python%20SDK%20References/deploy.md)**

沙箱资源部署管理器：
- `deploy_working_dir`：部署本地目录到沙箱
- `format`：模板变量替换，支持 `${working_dir}` 占位符
