# v1.2.5

## 发布日期
2026年2月28日

## 概述
本版本专注于监控和指标增强，新增支持自定义 OTLP 端点、用户自定义指标标签以及阿里云 MSE Nacos 连接。

---

## Admin

### 新特性

#### 自定义指标端点
- **新增**: 支持通过 `RuntimeConfig` 中的 `metrics_endpoint` 指定自定义 OTLP 指标端点。之前端点硬编码为 `http://{host}:{port}/v1/metrics`；现在可以配置为任何兼容 OTLP 的采集器。

**配置示例:**
```yaml
runtime:
  metrics_endpoint: "http://my-otel-collector:4317/v1/metrics"
```

#### 用户自定义指标标签
- **新增**: 支持通过 `RuntimeConfig` 中的 `user_defined_tags` 为所有指标添加自定义标签。这些标签会合并到 Admin、SandboxProxyService 和 SandboxActor 发出的每个指标的基础属性中。

**配置示例:**
```yaml
runtime:
  user_defined_tags:
    cluster: "prod-cluster"
    region: "cn-hz"
    service: "rock-sandbox"
```

#### 指标 IP 标签
- **新增**: 在指标基础属性中添加了 `ip` 字段，以便在监控仪表板中更好地识别实例。

#### Actor 指标端点
- **新增**: 将 `metrics_endpoint` 配置暴露给 `SandboxActor`，允许 Ray actors 向自定义 OTLP 端点报告指标。该端点在沙箱创建期间从 `RuntimeConfig` 传播到 actors。

### 增强功能

#### Nacos 配置
- **新增**: 在 `NacosConfig` 中添加了 `server_addresses` 字段，支持直接连接到 Nacos 服务器（例如阿里云 MSE），而无需仅依赖 `endpoint` 参数。

**配置示例:**
```yaml
nacos:
  server_addresses: "mse-xxx.nacos.mse.aliyuncs.com:8848"
  group: "DEFAULT_GROUP"
  data_id: "rock-config"
```
